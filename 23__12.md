### Задача 2.5

**Условие**: напечатать заполненный прямоугольный равнобедренный треугольник из `*` со сторонами катетов, равными N, и прямым углом справа снизу. где N — натуральное число больше 2.

```c++
int n;

cin >> n;

for (int i = 0; i < n; i++)
{
    for (int j = 1; j < n-1; j++)
    {
        cout << " ";
    }
    for (int j = 0; j <= i; j++)
    {
        cout << "*";
    }
    cout << endl;
}
```
